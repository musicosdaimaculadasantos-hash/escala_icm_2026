Guia de Visualização HTML

O HTML é a estrutura da web. Aqui estão as três formas principais de ver o que criaste:

🌐

No Navegador

Clica duas vezes no ficheiro .html guardado no teu computador para o abrir no Chrome ou Firefox.

✨

Botão Preview

Nesta plataforma, clica no botão "Preview" no canto superior do editor para ver o código em ação.

🛠️

Inspecionar

Numa página aberta, prime F12 ou clica com o botão direito e escolhe "Inspecionar" para ver o código fonte.

Este é um exemplo de como o código HTML se transforma numa interface visual.
